Bootstrap Basics

These are files I have made while learning to use Bootstrap. I'm using this site to learn: Tutorial Republic.

This page uses the Bootstrap framework and its default styles. (Personally, I find the default paragraph font-size too small, but you can change that.)

  • Hello, World! A basic template without a container DIV.
  • Template This one has the container.
  • Fixed layout
  • Fluid layout
  • Responsive columns These break down on smaller devices and screens exactly as you have spec'd them to break. They are 6 across on wide screens and then break down to 3, 2 and finally 1.
  • Navigation Bootstrap has built-in styles for both tabs and button navigation.
  • Typography Default styles for just about everything. Of course, you can change these.
  • Lists Several nice built-in styles.
  • Forms Bootstrap has lots of options for making HTML forms consistent and responsive.
  • Tables Nice defaults. Bootstrap is particularly nice if you need to generate a bunch of tables from dynamic data.
  • Grid example This example shows the responsive behavior of Bootstrap
  • Enhanced grid example More complex than the first grid example, here the grid adapts more completely to all screen sizes, with break points for large, medium and small screens.
  • Enhanced grid example with text This example is the same as the previous one, but with larger blocks of text inside each of the 12 divs, better demonstrating how the grid responds gracefully to different screen sizes (resize your browser window to see it in action).

Learning to use Bootstrap with the defaults seems the best way to start. You can, of course, modify anything by writing CSS or jQuery to do so. On a few of my pages, I added some simple CSS in the HEAD to increase some padding and/or margins.
